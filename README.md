# Emoji-Translator

## Overview
Emoji-Translator is a lightweight web application that translates standard text into corresponding emojis, or vice versa. This project is built using strictly front-end technologies to provide a fast, interactive, and user-friendly experience directly in the browser.

## Features
* **Real-Time Translation:** See the output update instantly as you type.
* **Responsive Design:** The layout adapts to different screen sizes, ensuring a clean interface on both desktop and mobile devices.
* **Lightweight:** No external libraries or frameworks are required.
* **Simple Interface:** Intuitive input and output fields for seamless use.

## Technologies Used
* **HTML5:** For the structure and layout of the application.
* **CSS3:** For styling, responsive design, and visual feedback.
* **JavaScript (Vanilla):** For the translation logic, event handling, and DOM manipulation.
